# Fahrieren (Trader Comm Platform)

A trading communication platform built with React 19, TypeScript 6, and Vite 7 (rolldown-vite).

## Tech Stack

- **React** 19.2
- **TypeScript** 6.0
- **Vite** 7.1 (rolldown-vite)
- **Deployment**: Static site on Hostinger

## Development

```bash
npm install
npm run dev      # Dev server
npm run build    # Production build
npm run lint     # ESLint
```
